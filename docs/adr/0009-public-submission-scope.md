# Limit public submissions to GitHub projects and SkillHub skills

`agi-radar submit` will accept public recommendations for GitHub projects and skills only.

Supported submission forms:

- `agi-radar submit github <github-repo-url>`
- `agi-radar submit skill <skillhub-cn-url>`

Skill submissions must use a `skillhub.cn` source URL. Article and news submissions are not accepted in the first public submission version because article/news quality and responsibility depend on external source sites and need a different review policy.

Submissions are review candidates. They do not become public signals until a publisher approves them and the publishing workflow emits them into the raw feeds.
